# 📊 Sales Performance Analysis Dashboard

## Overview

Developed an interactive Power BI dashboard to analyze sales performance, customer trends, and business KPIs. The project transforms raw sales data into meaningful insights to support business decision-making.

## Tools Used

* Power BI
* Excel
* Power Query
* DAX

## Key Features

* Sales & Revenue Analysis
* KPI Dashboard
* Product Performance Tracking
* Customer Insights
* Interactive Filters & Slicers
* Trend Analysis

## Skills Demonstrated

* Data Analysis
* Data Visualization
* Dashboard Development
* Data Modeling
* Business Intelligence
* KPI Reporting

## Outcome

Created a dynamic dashboard that helps stakeholders monitor business performance, identify sales trends, and make data-driven decisions.
